# Research Context Folder

This folder contains the full working context for the PhD research project on tomato detection and instance segmentation using TomatoMAP.

The goal of this folder is simple:
- Help a new reader understand the project quickly.
- Preserve the reasoning behind the chosen paper direction.
- Keep deadlines, constraints, and decisions visible.
- Avoid losing context when writing the paper, thesis, or discussing with a supervisor.

## Core project in one paragraph

This PhD project focuses on instance segmentation for greenhouse tomato phenotyping using the TomatoMAP dataset family. The immediate publication target is a first journal paper on label-efficient or semi-supervised instance segmentation using TomatoMAP-Seg, motivated by the current practical situation where only about 700 segmentation labels are available locally and the remaining 2k+ labels may or may not be released soon.

## Current working thesis direction

The thesis should be framed around **data-efficient tomato perception under annotation scarcity**, not around "yet another improved YOLO architecture".
