# Journal Strategy

## Primary target

### Computers and Electronics in Agriculture
This is the strongest first target because the journal scope covers machine vision, artificial intelligence, robotics, and agricultural applications. [web:83][web:100]

Why it fits:
- agricultural computer vision,
- practical segmentation,
- strong application framing,
- acceptable for a paper that is methodological but still grounded in crop phenotyping.

## Journal choice rule

Choose the journal based on how the paper reads:
- if it reads primarily like agricultural machine vision -> CEA.
- if it reads like plant-science enabling technology -> Frontiers.

## Current recommendation

Start writing toward **Computers and Electronics in Agriculture** first. [web:83][web:100]

## Submission mindset

Do not spend excessive time trying to make the paper fit multiple journal personalities at once. Write one strong version first.
