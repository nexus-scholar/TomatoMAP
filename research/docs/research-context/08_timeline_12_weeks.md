# Timeline: 12 Weeks to First Submission

## Week 1
- Audit files and annotations.
- Build a frozen list of labeled vs unlabeled images.
- Convert data to one training format only.
- Start writing project overview and problem statement.

## Week 2
- Create fixed train/val/test splits.
- Train first supervised baseline.
- Save first qualitative predictions.

## Week 3
- Stabilize the baseline.
- Start related work writing.
- Build the first result table draft.

## Week 4
- Freeze baseline protocol.
- Implement semi-supervised pipeline.
- Finish draft versions of Introduction, Related Work, and Dataset sections.

## Week 5
- Run first full semi-supervised experiment.
- Compare with supervised baseline.
- Start Methods section writing.

## Week 6
- Run label-fraction ablation.
- Record compute time and training stability.
- Add figure placeholders for pipeline and qualitative results.

## Week 7
- Perform error analysis.
- Write Results section draft.
- Build final tables and example figures.

## Week 8
- If missing labels arrive, run one controlled upper-bound experiment.
- If not, run one unlabeled-pool-size or robustness ablation.
- Finish complete manuscript draft.

## Week 9
- Revise novelty claims and tighten contribution list.
- Prepare journal-specific formatting.

## Week 10
- Get supervisor feedback.
- Run only missing experiments that answer obvious reviewer questions.

## Week 11
- Finalize discussion, limitations, and data availability sections.
- Prepare cover letter and supplementary material.

## Week 12
- Final proofreading.
- Submit first article.
- Immediately begin thesis chapter adaptation from the submitted paper.
